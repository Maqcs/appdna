# ApplicationGroupHierarchyRoot Class
 

Represents the root of the <a href="2eac72dc-6d12-130d-75ef-83c92c9c4bfe">ApplicationGroup</a> tree. Use this as a data binding object to display and monitor the <a href="2eac72dc-6d12-130d-75ef-83c92c9c4bfe">ApplicationGroup</a> tree and stay up to date while the hierarchy is modified using the SDK. Retrieve it by calling <a href="162301f8-fd8f-9717-0b41-e9be385fa4ab">GetApplicationGroupHierarchyRoot()</a>.
